### Added

- **New project** command imports TextGrid files in UTF-8, UTF-16 and Latin-1
- **Find** command
- **Find and Replace** command
- **Map annotations**
- **Sort table by column (A to Z)** sorts columns in ascending  order
- **Sort table by column (Z to A)** sorts columns in descending order
- **Filter by** filters rows dynamically.
- **Open selection in Praat** opens the selected row.
- **Preferences** command manages the application settings.
